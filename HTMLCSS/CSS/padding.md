# padding, margin

```
The box has 4 things.

1. contents
2. padding / inside the border box
3. border
4. margin / outside the border box


padding means space from the border and it's inside
padding is pushing to the inside.
margin is pushing to the outside.

padding: 20px // everywhere
padding: 20px(top,bottom) 10px(right,left) // clock wise.

border-width: 5px;
border-style: dashed;
border-color: red;
// short cut => border: 5px dashed red;
```
